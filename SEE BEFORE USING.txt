=> 💻 HOW TO RUN THIS ON YOUR PC (LOCAL DEVELOPMENT)
--------------------------------------------------------------------------------
If you are a developer or supervisor who wants to run the source code on your local machine, 
please follow these exact steps:

PREREQUISITES:
1. You must have Node.js installed (v18+ recommended).
2. You must have an OpenAI API Key.

STEP 1: DOWNLOAD & EXTRACT
Download the repository or clone it via GitHub.
Navigate your terminal into the root folder: `cd beastlife-support`

STEP 2: ENVIRONMENT VARIABLES
In the root `beastlife-support` directory, there is a file named `.env.example`.
Rename this file to `.env` (or create a new file named `.env`).
Open it and paste your OpenAI API Key:

OPENAI_API_KEY=sk-your-real-api-key-goes-here
PORT=3001

STEP 3: INSTALL ALL DEPENDENCIES
Run the following command in your terminal from the root folder:
`npm run install:all`
(This will concurrently install the root, frontend, and backend packages).

STEP 4: START THE APPLICATION
Run the following command:
`npm run dev`

This will boot up both systems concurrently:
- The Backend Engine will start on `http://localhost:3001`
- The Frontend Application will start on `http://localhost:5173` (or 5174/5175 if busy)

Your terminal will display the exact Local URL. Hold 'Ctrl' and click the URL to open it.
